# ⚙️ Playground TS Generation

Aplicação desenvolvida em **React + TypeScript** com foco em experimentação de conceitos de geração de código, tipagem e organização de componentes.

O projeto funciona como um ambiente de testes (*playground*), permitindo explorar abordagens modernas de desenvolvimento front-end com ênfase em boas práticas e escalabilidade.

---

## 🎓 Contexto acadêmico

Este projeto foi desenvolvido com fins acadêmicos, com o objetivo de:

* Explorar recursos avançados do TypeScript
* Praticar componentização com React
* Testar padrões de organização de código
* Simular cenários de geração e manipulação de dados

---

## 🚀 Tecnologias utilizadas

* ⚛️ React
* 📘 TypeScript
